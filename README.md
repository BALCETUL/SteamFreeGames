# SteamFreeGames

**SteamFreeGames** — проект для автоматического сбора и отображения всех игр со скидкой **100%** в магазине Steam.

## Функции

- Краулер на Python (NeedFree.py) периодически (расписание в GitHub Actions) собирает данные.
- Хранит результат в free_goods_detail.json.
- Простая веб-страница index.html для просмотра списка бесплатных игр.
- Автоматический коммит и пуш обновлённого JSON.

## Как это работает

1. **GitHub Actions** (.github/workflows/python-app.yml) каждый день в 00:00 UTC запускает скрипт.
2. Скрипт NeedFree.py собирает список игр с 100% скидкой.
3. Результат сохраняется в free_goods_detail.json.
4. Веб-страница index.html динамически подгружает этот JSON и отображает его.

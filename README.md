# SteamFreeGames 🎮

**SteamFreeGames** — это автоматизированный проект по сбору и отображению "ИГР и DLC" со 100% скидкой в Steam.  
Проект автоматически обновляется каждый час и отображает список бесплатных "ИГР и DLC на веб-странице.

🔗 **Сайт:** [SteamFreeGames](https://balcetul.github.io/SteamFreeGames/)  
🔁 Обновление: каждый час (могут быть задержка от сервера) (UTC+3)  
📄 JSON-данные: [free_goods_detail.json](./free_goods_detail.json)

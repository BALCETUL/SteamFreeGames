teamFreeGames 🎮
SteamFreeGames — это автоматизированный проект для сбора и отображения игр с 100% скидкой в Steam.
Данные обновляются ежедневно автоматически, отображаются на сайте в виде списка.

🔗 Сайт: SteamFreeGames
🕒 Обновление: каждый день в 00:00 по Киеву (Europe/Kiev, UTC+3)
📄 JSON-данные: free_goods_detail.json

📌 Возможности
Автоматический сбор игр с 100% скидкой со страницы поиска Steam.

Ежедневное обновление с помощью GitHub Actions.

Простая и удобная веб-страница на Bootstrap 5.

JSON-данные с количеством и временем последнего обновления.

⚙ Как это работает
GitHub Actions (.github/workflows/python-app.yml) запускает краулер каждый день в 00:00 по Киеву.

Скрипт NeedFree.py парсит Steam и сохраняет данные в free_goods_detail.json.

Веб-страница index.html подгружает JSON и показывает список бесплатных игр.

Все изменения автоматически коммитятся и публикуются.
